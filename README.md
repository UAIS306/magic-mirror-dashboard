# magic-mirror-dashboard
Dashboard for "Magic Mirror" project. Displays weather, transit, news and other info.

## Setup

1. install node & npm
2. install ruby
3. sudo npm install -g grunt
4. gem install bundler
5. npm install
6. bundle install
7. `grunt build`
8. `node index.js`
